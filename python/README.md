# stockseyes (Python) — 🚧 Coming soon

The Python SDK for the [Stockseyes](https://github.com/stockseyes/stockseyes-package) APIs is planned and will be published to [PyPI](https://pypi.org/) as `stockseyes`.

It will conform to the same contract as every other Stockseyes SDK — see the root
[`spec/openapi.yaml`](../spec/openapi.yaml) and the shared [`fixtures/`](../fixtures), which
this package's tests will reuse so its behaviour matches `@stockseyes/node` exactly.

In the meantime, use the Node.js SDK: [`@stockseyes/node`](../node).
